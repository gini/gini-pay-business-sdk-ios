Gini Pay Business SDK for iOS

The Gini Pay Business SDK provides components for uploading, reviewing and analyzing photos of invoices and remittance slips.

By integrating this SDK into your application you can allow your users to easily upload a picture of a document, review it and get analysis results from the Gini backend, create a payment and send it to the prefferable payment provider.

Documentation

Further documentation with installation, integration or customization guides can be found in our website.

Example

We are providing example app for Swift. This app demonstrates how to integrate the Gini Pay Business SDK with the Component API of Gini Capture library. To run the example project, clone the repo and run pod install from the Example directory first. To check the redirection to Banking app please run Bank example before Example Swift. Requirements

  • iOS 10.2+
  • Xcode 10.2+

Note: In order to have better analysis results it is highly recommended to enable only devices with 8MP camera and flash. These devices would be:

  • iPhones with iOS 10.2 or higher.
  • iPad Pro devices (iPad Air 2 and iPad Mini 4 have 8MP camera but no flash).
